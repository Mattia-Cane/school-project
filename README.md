# school-project
needed for a school project


This site is meant to generate users and show them in a grid-like view. in the navBar you can choose the Home, which is the page you're on at the start, the Genera utenti that allows you to generate the users via a slider, that lets you decide how many users to generate, radio buttons that let you wihich gender the people should be and the checkboxes wich allows you to choose which nation should be included. There's also a button that let's you increase the maximum generaed user ammount to one thousand but you can access it only after reaching the slider maxium user generation, which is 100 by default. the Genereate user button allows you to generate and view the users in the Table view section, wich are 10 per row. There's a detail button which it'll be implemented in the future.
